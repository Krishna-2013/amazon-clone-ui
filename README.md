# Amazon Clone UI

A front-end clone of the Amazon homepage built using HTML and CSS.
This project focuses on replicating a real-world e-commerce UI layout.

## ✨ Features

* Amazon-style homepage layout
* Navigation bar with logo, search bar, and menu
* Product grid layout
* Footer section similar to Amazon
* Built using Flexbox/Grid

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🎯 Purpose

This project was built to improve front-end development skills by recreating a real-world interface.

Key learning areas:

* Layout structuring
* Spacing and alignment
* UI design patterns

## 💡 Key Challenges

- Creating complex layout similar to Amazon
- Managing spacing and alignment
