# Amazon Clone UI

## Description

This project is a front-end clone of the Amazon homepage built using HTML and CSS.
The goal of this project was to practice layout design, positioning, and styling by recreating a real-world website interface.

## Features

* Amazon-like homepage layout
* Navigation bar with logo, search bar, and menu items
* Product sections and grid layout
* Footer design similar to the original site
* Styled using pure CSS (Flexbox and/or Grid)

## Technologies Used

* HTML5
* CSS3

## Purpose of This Project

This is a practice project created to improve front-end development skills by replicating an existing UI.

It focuses on:

* Layout structuring
* Spacing and alignment
* Real-world UI design patterns

## Disclaimer

This project is only for educational purposes.
It is a clone of the Amazon homepage and is not intended for commercial use.
